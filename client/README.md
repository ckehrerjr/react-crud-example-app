# ReactJS - Crud Example App
> An example of a crud application that integrates api data with Redux state management using it's lifecycle process.

## Technologies
> Project is created with:  
  * React.js 16
  * ReactDOM 16
  * React Router 4.3.1
  * Lodash 4
  * Redux 4
  * Redux Thunk 2.3
  * Redux Form 8.0.4
  * Json Server 0.14

## Setup
```
$ npm install
$ npm run dev

$ yarn install
$ yarn start
```
